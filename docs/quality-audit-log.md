# 일일 발행 퀴즈 품질 검수 로그

매일 발행분(카테고리 5문항)을 전수 검수하고 결과를 기록한다.
모든 판정은 quiz id 기준 — DB(`quiz`/`choice`)와 실험 테이블(`trial_quiz`)로 재검증 가능.

## 판정 기준
- **치명 결함**: 복수 정답(참인 오답 포함), 전제-정답 내적 모순, 정답 방향 오류, 자립성 위반(기사 없이는 성립 불가)
- **경계**: 오답이 논쟁적이거나 부분 참 소지가 있으나 질문 정합 기준으로는 정답이 유일 최선
- **관찰**: 결함은 아니지만 반복되면 조치할 패턴 (소재 중복, 필터 구멍 등)
